Before You Begin!

  • Make sure your main image is saved as MainImage.png , in the same directory as createimage.html
  • If you want to change the icon that is used to point to places on your image, save it as info_icon.png in the Tooltipster/images directory

Directions

  • In the table below, specify the x and y coordinates (in pixels) of the tip as well as the html content
  • The tip number must go in ascending order (1, 2, 3, etc)
  • After you have entered all the information, click the Store Data button
